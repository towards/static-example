Hello static World from Gleis!

Feel free to re-use the structure of this simple static website as an example in order to build your own static site for the Gleis Platform as a Service.

The public directory contains the root of your static site and the assets sub-directory is used to store your static assets such as images. The whole directory tree looks like this:

  • public
    • assets
      • images
      • javascripts
      • stylesheets

You will find more details on how to deploy your static site on the GitHub page of this example static site.
